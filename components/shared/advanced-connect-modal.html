Connect with Private Key

If you are using MetaMask, check how to export your private key guide. If you are using another wallet, please refer to their documentation.

Exporting and using private keys directly in the browser poses major security risks. Client-side storage or exposure of private keys may result in unauthorized access. Proceed only if you fully understand and accept these risks.
  • Never use this on a public computer.
  • Only use on a trusted browser with only trusted extensions.
  • Always backup your private key securely.
  • Ensure you are familiar with the risks of managing your own private keys.
  • Your private key will be persisted on disk in the end (by the browser), so any malicious access could compromise your security.
